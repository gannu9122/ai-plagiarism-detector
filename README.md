# 🧠 AI Plagiarism Detector

An AI-powered web application that detects plagiarism between multiple PDF documents using **TF-IDF vectorization**, **cosine similarity**, and **sentence-level matching**, along with an interactive dashboard built using Flask and Bootstrap.

---

## 🚀 Features

- 📄 Upload multiple PDF files
- 🧠 AI-based text extraction and analysis
- 🔍 Sentence-level similarity detection
- 📊 Hybrid similarity scoring system
- 🎨 Color-coded plagiarism risk levels:
  - 🟢 Low (0–40%)
  - 🟡 Medium (40–70%)
  - 🔴 High (70–100%)
- 📈 Interactive dashboard with progress bars
- 🔎 Expandable matched sentence details
- ⚡ Optimized TF-IDF matrix computation for performance

---

## 🛠️ Tech Stack

- Python 🐍
- Flask 🌐
- Scikit-learn 🤖
- HTML / CSS / Bootstrap 🎨
- PDF Processing Libraries (PyPDF / custom extractor)
